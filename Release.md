# Release Notes - FufuLauncher v1.4.1

## 🎉 First Release

Welcome to the first public release of **FufuLauncher** (芙芙启动器) — a third-party Genshin Impact launcher built with WinUI 3.

---

## 🚀 Key Features

### 🔐 Account Management
- Multi-account quick switching with encrypted local storage
- MiHoYo QR code login support

### 📅 Auto Check-in
- One-click daily MiHoYo/HoYoLab check-in
- Cloud game check-in support
- Auto community tasks (likes, reads, shares)

### 🎮 Game Management
- Automatic game path detection
- Real-time version update announcements
- Game resource pre-download & integrity verification
- Server switching (Official / Bilibili / Global)

### ⚡ Utility Tools
- **Cultivation Calculator** — calculate character/weapon upgrade resources
- **Auto Clicker** — keyboard/mouse automation
- **FPS Monitor** — real-time frame rate display
- **Screenshot Tool** — one-click game capture
- **Wish History Viewer** — gacha data analysis
- **Achievement Tracker** — track achievement progress

### 🚀 Launch Parameters
- Custom resolution & window mode presets
- Multi-monitor support

### 🔧 Injection System
- DLL injection with preset management
- Plugin extension support

### 🎨 Customization
- Custom backgrounds (image/video/slideshow)
- Theme colors & Acrylic/Mica effects
- Adjustable opacity

---

## 🐛 Known Issues

- Dynamic video backgrounds may be unstable; static images are recommended
- Injection features require Administrator privileges
- Some features may behave differently on Global (OS) vs CN server clients

---

## 📥 Installation

### System Requirements
- **OS**: Windows 10/11 (64-bit)
- **Runtime**: .NET 8.0 or later
- **Runtime**: Microsoft Edge WebView2
- **Runtime**: Visual C++ Redistributable v14
