# Tùy Chỉnh Nền

## Hướng Dẫn Nhanh

1. Nhấn biểu tượng **⚙️ Cài Đặt** ở góc dưới bên trái
2. Vào mục **Nền & Hiệu Ứng**
3. Bật **Bật nền tùy chỉnh**
4. Nhấn **Chọn ảnh nền...** và chọn file ảnh/video

## Mẹo & Lưu ý

- Nên dùng ảnh có độ phân giải tương đương màn hình
- Hỗ trợ: ảnh (PNG, JPG) và video (MP4)
- Nền video động có thể không ổn định
- Nếu chưa hiển thị, thử khởi động lại với quyền Quản trị Viên
- Tắt công tắc để khôi phục nền mặc định

## Các Tùy Chọn Nâng Cao

### Slideshow Nền
Bật chế độ slideshow để tự động chuyển đổi nhiều ảnh nền:
1. Bật **Slideshow nền**
2. Chọn thư mục chứa ảnh
3. Cài đặt thời gian chuyển ảnh (giây)

### Độ Trong Suốt
Điều chỉnh độ trong suốt của các lớp nền trong cài đặt.

### API Nền Tùy Chỉnh
Nếu bạn có API nền riêng, có thể nhập URL tùy chỉnh trong phần cài đặt nâng cao.
