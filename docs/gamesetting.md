# Hướng Dẫn Sử Dụng

## Tổng Quan Giao Diện

FufuLauncher là công cụ hỗ trợ khởi động game Genshin Impact của bên thứ ba, hỗ trợ quản lý đường dẫn game, chuyển đổi tài khoản và máy chủ. Giao diện chính chia làm 3 module:

- **Khu vực cài đặt game**: Quản lý đường dẫn game, trạng thái client
- **Khu vực chuyển đổi tài khoản**: Thêm/xoá/chuyển đổi tài khoản
- **Khu vực chức năng**: Quản lý tài nguyên, chuyển máy chủ, bản đồ

## Cài Đặt Cơ Bản

### 1. Thiết Lập Đường Dẫn Game
1. Tìm module **Vị trí cài đặt game**
2. Nhấn **Duyệt**, chọn thư mục chứa `Genshin Impact Game`
3. Nhấn **Áp dụng** để lưu

### 2. Xem Trạng Thái Client
Xem trực tiếp trên giao diện:
- Phiên bản hiện tại
- Máy chủ
- Dung lượng ổ đĩa
- Phiên bản mới nhất
- Trạng thái tài nguyên tải trước

> **Mẹo**: Phiên bản không khớp sẽ không khởi động game được.

## Chuyển Đổi Tài Khoản

1. **Thêm tài khoản**: Nhấn `+` ở góc phải module tài khoản
2. **Chuyển tài khoản**: Nhấn nút **Chuyển** tương ứng
3. **Xoá tài khoản**: Nhấn biểu tượng thùng rác
4. **Làm mới**: Nhấn biểu tượng làm mới

## Tính Năng Chính

### 1. Quản Lý Tài Nguyên
- Chức năng: Kiểm tra, sửa chữa, cập nhật file game
- **Khuyến nghị**: Dùng trình khởi động chính thức của MiHoYo để cập nhật

### 2. Chuyển Đổi Máy Chủ
- Hỗ trợ: CN / B-CN / Quốc tế
- **Lưu ý**:
  1. **Sao lưu file game** trước khi chuyển
  2. Nếu thất bại, vào nhóm để xem hướng dẫn
  3. Máy chủ Quốc tế có thể có lỗi âm thanh

### 3. Bản Đồ Tương Tác
Nhấn **Bản đồ** để xem vị trí rương, mắt thần, nhiệm vụ.

### 4. Tạo Lối Tắt
Tạo shortcut trên desktop để khởi động game trực tiếp.

### 5. Mã Đổi Thưởng
Xem và sao chép mã đổi thưởng mới nhất từ chính thức.

## Lưu Ý Quan Trọng

1. Đảm bảo **đường dẫn game chính xác**, tránh ký tự đặc biệt
2. **Không chuyển máy chủ quá thường xuyên** - có rủi ro hỏng file
3. Tài khoản chỉ lưu cục bộ, **không lưu trên máy công cộng**
4. FufuLauncher là công cụ hỗ trợ, **không sửa file game**

## Xử Lý Sự Cố

1. "Không tìm thấy file game": Thiết lập lại đường dẫn
2. Chuyển tài khoản thất bại: Kiểm tra mạng, làm mới danh sách
3. Phiên bản không khớp: Dùng trình khởi động chính thức để cập nhật
4. Lỗi âm thanh máy chủ Quốc tế: Dùng trình khởi động MiHoYo sửa chữa
