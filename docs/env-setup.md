# Cài Đặt Môi Trường

## Yêu Cầu Hệ Thống

Để chạy FufuLauncher, bạn cần cài đặt các thành phần sau:

### 1. .NET Runtime

1. Mở trình duyệt, truy cập [.NET 8.0](https://dotnet.microsoft.com/download/dotnet/8.0)
2. Tìm phiên bản dành cho Windows - chọn **x64 Installer**
   ![.NET Download](https://fu1.fun/image/image-1.png)
3. Đợi quá trình tải xuống hoàn tất (có thể dùng accelerator để tăng tốc)
   ![Installing](https://fu1.fun/image/image-2.png)
4. Chạy file cài đặt và làm theo hướng dẫn
   - Nếu hiển thị "Modify installation" nghĩa là bạn đã có .NET, chỉ cần đóng lại

> **Lưu ý**: Một số người dùng báo cáo .NET 8.0 không hoạt động, hãy thử [.NET 10.0](https://dotnet.microsoft.com/download/dotnet/10.0)

### 2. Visual C++ Redistributable v14

1. Truy cập [Visual C++ Redistributable](https://learn.microsoft.com/cpp/windows/latest-supported-vc-redist)
2. Tải xuống phiên bản **X64**
   ![VC++ Download](https://fu1.fun/image/image-4.png)
3. Chạy file cài đặt và làm theo hướng dẫn

### 3. Microsoft Edge WebView2

1. Truy cập [Microsoft Edge WebView2](https://developer.microsoft.com/microsoft-edge/webview2/)
2. Tìm phần **Evergreen Standalone Installer** và nhấn **Download**
   ![WebView2 Download](https://fu1.fun/image/image-3.png)
3. Chạy file cài đặt và đợi hoàn tất

## Kiểm Tra Sau Khi Cài Đặt

Sau khi cài đặt đầy đủ, bạn có thể chạy FufuLauncher ngay. Nếu gặp lỗi:

- Kiểm tra lại xem đã cài đủ các thành phần chưa
- Thử chạy với quyền Quản trị Viên
- Vào [Discord](https://discord.gg/fvrYwyCR) để được hỗ trợ
